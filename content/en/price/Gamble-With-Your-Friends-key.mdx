|---|---|---|
| **Game key** | A code that unlocks ownership of the game | You need a valid code to activate |

| **Region lock** | Key is limited to certain countries | Buying the wrong region causes redemption failure |
| **Edition key** | Standard/Deluxe/Bundle license | Different content and price points |
| **Reseller stock** | Key sourced from third-party supply | Price can be lower, risk can be higher |

When comparing listings for a **Gamble With Your Friends key**, verify all four items before paying:

| **“Invalid product code”** | Typo, wrong launcher, broken code | Paste again, verify platform, contact seller with screenshot |
